### 近期趋势条形图

```swift
// 使用 Swift Charts，非常简单
struct VPIPTrendChart: View {
    let data: [DayVPIP]  // 最近 5-7 天

    var body: some View {
        Chart(data) { day in
            BarMark(
                x: .value("VPIP", day.vpip),
                y: .value("Day", day.label)
            )
            .foregroundStyle(day.isHigh ? .warning : .pokerGreen)
        }
        .chartXScale(domain: 0...40)
        .frame(height: 140)
    }
}

struct DayVPIP: Identifiable {
    let id = UUID()
    let label: String   // "周一"
    let vpip: Int       // 19
    var isHigh: Bool { vpip > 25 }
}
```

### 玩家类型说明

| VPIP | 类型 | 描述 |
|------|------|------|
| < 15% | 极紧玩家 | 只玩最强牌 |
| 15-19% | 紧凶玩家 | 选择性入池 |
| 20-24% | 标准玩家 | 平衡风格 |
| 25-29% | 松凶玩家 | 范围较宽 |
| ≥ 30% | 极松玩家 | 入池过多 |

## Pro 功能处理

### 策略：数据预留，UI 不显示

```
MVP 阶段：
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
- 数据库字段预留 position, bbResult
- UI 不显示任何 🔒 锁定功能
- 不显示 Pro 弹窗
- 用户感知：这是一个完整的免费 App

后续版本：
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```
