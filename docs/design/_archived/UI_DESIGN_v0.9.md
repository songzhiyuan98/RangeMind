# TiltGuard UI Design System v1.0

> iOS 26 Liquid Glass Design Language

---

## 设计理念

### 核心原则

**1. 液态玻璃 (Liquid Glass)**
- 半透明毛玻璃材质
- 动态光影折射
- 深度层次感
- 环境自适应色彩

**2. 极简交互**
- 单手可操作
- 最少点击完成记录
- 大触控区域
- 即时视觉反馈

**3. 沉浸式体验**
- 全屏设计，减少边框
- 内容优先，UI 退居幕后
- 流畅动画过渡
- 触感反馈 (Haptics)

### 语义色彩

| 用途 | Light Mode | Dark Mode | 场景 |
|------|------------|-----------|------|
| Success | `#00C853` | `#69F0AE` | WIN、正常状态 |
| Warning | `#FF9100` | `#FFAB40` | Tilt 警告 |
| Danger | `#FF1744` | `#FF5252` | 严重警告、NOT WIN |
| Info | `#2979FF` | `#448AFF` | 提示信息 |

### 玻璃材质色

```swift
// 液态玻璃背景
struct LiquidGlass {
    // 主要玻璃层
    static let primary = Color.white.opacity(0.15)

    // 次要玻璃层（更透明）
    static let secondary = Color.white.opacity(0.08)

    // 强调玻璃层（更实）
    static let elevated = Color.white.opacity(0.25)

    // 模糊半径
    static let blurRadius: CGFloat = 20

    // 边框
    static let border = Color.white.opacity(0.3)
}
```

### 背景渐变

```swift
struct LiquidGlassCard: ViewModifier {
    func body(content: Content) -> some View {
        content
            .background(.ultraThinMaterial)
            .background(Color.white.opacity(0.15))
            .clipShape(RoundedRectangle(cornerRadius: 24, style: .continuous))
            .overlay(
                RoundedRectangle(cornerRadius: 24, style: .continuous)
                    .stroke(Color.white.opacity(0.3), lineWidth: 1)
            )
            .shadow(color: .black.opacity(0.1), radius: 24, y: 8)
    }
}
```

## 动画规范

### 过渡动画

| 场景 | 动画类型 | 时长 | 曲线 |
|------|----------|------|------|
| 页面切换 | Slide + Fade | 0.3s | easeInOut |
| 卡片展开 | Scale + Fade | 0.25s | spring(0.7) |
| 按钮点击 | Scale Down | 0.1s | easeOut |
| 弹窗出现 | Bottom Sheet | 0.35s | spring(0.8) |
| 数字变化 | Counter | 0.4s | easeOut |

### 触感反馈

| 操作 | 触感类型 |
|------|----------|
| FOLD 点击 | `.impact(.light)` |
| VPIP 点击 | `.impact(.medium)` |
| 卡牌选择 | `.selection` |
| 结果确认 | `.impact(.medium)` |
| Tilt 警告 | `.notification(.warning)` |
| Session 结束 | `.notification(.success)` |

### 微交互

```swift
// 按钮按压效果
.scaleEffect(isPressed ? 0.96 : 1.0)
.animation(.easeOut(duration: 0.1), value: isPressed)

// 卡牌选中效果
.scaleEffect(isSelected ? 1.1 : 1.0)
.overlay(
    RoundedRectangle(cornerRadius: 12)
        .stroke(Color.green, lineWidth: isSelected ? 3 : 0)
)
.animation(.spring(response: 0.3, dampingFraction: 0.7), value: isSelected)

// VPIP 数字变化
.contentTransition(.numericText())
```
